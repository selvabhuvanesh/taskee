Getting Started

taskoot is a family task management app where parents can assign tasks, set rewards, and children can earn coins by completing them.

  • Parents: Sign in with Apple, create a family, and invite members using your family code
  • Children: Join your family using the code shared by your parent

Frequently Asked Questions

How do I create a family?

After signing in, choose the "Parent" role. A family code will be generated automatically. Share this code with family members to join.

How do children join?

Children sign in with Apple (or use a parent's device), choose the "Child" role, and enter the family code.

How does data sync work?

taskoot uses Apple iCloud (CloudKit) to sync data across family members' devices. All members need to be signed into iCloud on their devices.

What are coins?

Coins are rewards children earn for completing tasks. Parents set the reward amount when creating tasks. Children can request to redeem coins for real-world rewards.

How do subscriptions work?

taskoot offers a free tier with basic features. Family and Pro tiers unlock additional features like unlimited tasks, templates, and more.
